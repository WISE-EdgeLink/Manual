# 网关与三菱L02 PLC的网口通讯
**PLC端的设置**
PLC软件：GX Works 2，设置PLC的IP地址和端口号。
注意事项：
1、每条链接的端口号可自由设置，但不能相同，；
2、每条链接只允许一个访问终端使用，如果有多个访问终端，需要配置多条链接；
3、配置下载到PLC后需要断电重启PLC。
