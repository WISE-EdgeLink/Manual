### IO点搜索

除按tag名称进行搜索外，IO点页面还支持根据端口和仪表进行筛选

![](TS_002_cn.png)

![](TS_003_cn.png)

|  按钮   |  描述  |
|  ----  | ----  |
| 筛选  | 在端口列表中进行勾选后，点击筛选按钮方可生效 |
| 重置  | 点击重置按钮则取消所有勾选，即恢复为不筛选的状态 |
